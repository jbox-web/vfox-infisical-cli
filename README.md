# vfox-infisical-cli

[![CI](https://github.com/jbox-web/vfox-infisical-cli/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-infisical-cli/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [infisical-cli](https://github.com/Infisical/cli)

## Install

```
mise plugin install infisical-cli https://github.com/jbox-web/vfox-infisical-cli.git
```
